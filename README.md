# HotCRP Watch

Generated by Codex + GPT-5.5.

Headless helper for inspecting and watching specified HotCRP paper pages; it prompts for your password in the terminal, stores only the session cookie under `state/`, and saves changed pages as HTML, text, JSON, and diff files under `state/paper-<id>/`.

## Usage

Inspect once:

```bash
python3 hotcrp_watch/hotcrp_watch.py inspect --site https://iccad2026.hotcrp.com --email your@email.edu --paper 50,51
```

Watch every 60 seconds:

```bash
python3 hotcrp_watch/hotcrp_watch.py watch --site https://iccad2026.hotcrp.com --email your@email.edu --paper 50,51 --interval 60
```

Watch with PushPlus ([official site](https://www.pushplus.plus/)):

```bash
export PUSHPLUS_TOKEN='your-pushplus-token'
python3 hotcrp_watch/hotcrp_watch.py watch --site https://iccad2026.hotcrp.com --email your@email.edu --paper 50,51 --interval 60
```

Optional flags: use `--reset-session` to force a fresh login, `--url <url1>,<url2>` instead of `--paper`, `--pushplus-token <token>` instead of `PUSHPLUS_TOKEN`, and `--pushplus-topic <topic>` or `PUSHPLUS_TOPIC` for PushPlus groups.

## Boundary

The tool only reads pages returned to your logged-in session and only posts to `/signin` for login; it does not submit, withdraw, edit papers, query unrelated endpoints, or recover data not returned by the server.
